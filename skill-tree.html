⭐ Prestige-System (Level 99+)

Bei Erreichen von Level 99 wird 99+ angezeigt. Das Mitglied kann in den Prestige-Modus wechseln und eine permanente Funktion für immer freischalten – solange der Account existiert.

⭐ Prestige-Modus – Permanente Freischaltungen

Jedes Prestige-Level schaltet eine permanente Funktion frei. Das Level wird auf 0 zurückgesetzt, aber alle freigeschalteten Features bleiben für immer erhalten. Der Prestige-Zähler wird im Profil neben dem Level angezeigt.

Prestige I

💎 Exklusives Profil-Design

Animated Profile Border · Benutzerdefinierte Farbschemata · Exklusive Profilrahmen die nur Prestige-Mitglieder haben

Prestige II

📈 Erweiterte Analytik

Zugang zu detaillierten Statistiken über eigene Aktivität · Fortgeschrittene Insights · Export-Funktion

Prestige III

💰 Premium-Marktforschung

Zugang zu exklusiven, hochdotierten Umfragen · 2x Gutschein-Multiplikator · Frühzeitiger Zugang

Prestige IV

🌱 Mentor-Suite

Eigenes Mentor-Dashboard · Sichtbar als offizieller Mentor · Priorität bei Matching · Mentoring-Badges

Prestige V

🌟 Featured Status

Dauerhafter „Featured Researcher”-Status · Priorisierte Sichtbarkeit in Suchergebnissen und Empfehlungen

Prestige VI

🧠 Content-Creation Studio

Erweiterte Tools zum Erstellen von Quizzen, Umfragen und interaktivem Content · Eigene Template-Bibliothek

Prestige VII

🔒 API-Zugang

Persönlicher API-Zugang für Datenexport · Eigene Integrationen · Entwickler-Tools

Prestige VIII

🏛️ Eigene Community

Erstelle eigene Sub-Communities · Private Foren · Eigene Einladungscodes · Community-Management-Tools

Prestige IX

👑 VIP-Netzwerk

Zugang zum exklusiven VIP-Netzwerk · Direkte Kontakte zu Industriepartnern · Exklusive Events

Prestige X – MAX

🐐 GOAT – Hall of Fame

Permanenter GOAT-Status · Hall of Fame Eintrag · Alle Features freigeschalten · Eigene Challenges für die Community erstellen

⚙️ Technische Konfiguration

XP-Formel pro Level

// XP benötigt für Level n
xp_required(n) = floor(100 * (1.08 ^ n) + 50 * n)

// Beispiele:
Level 1: 158 XP · Level 10: 716 XP · Level 25: 2.592 XP
Level 50: 14.689 XP · Level 75: 76.914 XP · Level 99: 384.951 XP

// Gesamt-XP für Level 99:
Total: ~4.850.000 XP

Score-Rarity Mapping

// Jede Einreichung speichert:
{ score: 0–25, rarity: "common"|...|"mythical", is_goat: boolean }

Common: score 0–5
Uncommon: score 6–10
Rare: score 11–15
Epic: score 16–20
Legendary: score 21–23
Mythical: score 24–25
GOAT: is_goat = true (manuell / HoF)

// Profil zeigt nur den höchsten Wert (stackbar)
displayed_rarity = max(all_submissions.score)

Prestige XP-Multiplikator

// Multiplikator pro Prestige-Stufe
multiplier(p) = 1.0 + (p * 0.15)

Prestige I: 1.15x · II: 1.30x · III: 1.45x · IV: 1.60x · V: 1.75x
Prestige VI: 1.90x · VII: 2.05x · VIII: 2.20x · IX: 2.35x · X: 2.50x

Streak-Bonus-Formel

// Täglicher Streak-Bonus
streak_xp(day) = 20 + min(day * 5, 200)

Tag 1: 25 XP · Tag 7: 55 XP · Tag 30: 170 XP · Tag 40+: 220 XP (cap)
